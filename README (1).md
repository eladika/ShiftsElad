## ✨ פיצ'רים

- 📅 לוח שבועי ראשון–שישי עם בוקר וערב
- 🏠 סימון WFH (עבודה מהבית) בצבע כחול
- 🎨 קידוד צבעים: בוקר / WFH / ערב / חופש / מחלה / מילואים
- ⚠️ ולידציה: אזהרות על פחות מ-5 משמרות או יותר מ-WFH אחד
- 💾 שמירה אוטומטית ב-localStorage
- 📋 תורנויות שבועיות
- 📊 טאב סיכום עם סטטיסטיקות
- ⚙️ פאנל ניהול: הוספה/עריכה/מחיקת עובדים, גיבויים
- 📥 ייצוא Excel
- 🖨️ הדפסה
- 🔁 שכפול שבועות
- 🇮🇱 עברית מלאה RTL

## 🔧 שינוי נתוני עובדים

ערוך את `RAW_EMPLOYEES` בתוך הקובץ:

```javascript
{ id:1, name:'שם העובד', ext:'700', role:'adv', backup:'שם המגבה',
  sched:{ 0:{m:'morning',e:'off'}, ... } }
```

- `role`: `mgr` / `adv` / `int`
- `m` / `e`: `morning` / `wfh` / `evening` / `vacation` / `sick` / `reserve` / `off`
